🔗 Conexión con Google Sheets

📋 Pasos para conectar:

  1. Abre script.google.com y crea un nuevo proyecto
  2. Pega el contenido de Code.gs (archivo descargado)
  3. Reemplaza SPREADSHEET_ID con el ID de tu Google Sheet
  4. Click en Implementar → Nueva implementación → Aplicación web
  5. Ejecutar como: Yo · Acceso: Cualquier usuario
  6. Copia la URL del Web App y pégala arriba
  7. Click en "Inicializar hojas" para crear las tablas automáticamente
